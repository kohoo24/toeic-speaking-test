## 🔄 워크플로우

### 1단계: 사용자 등록
1. 사용자 관리 → 엑셀 업로드
2. 엑셀 형식: A열(이름), B열(수험번호)

### 2단계: 문제 생성
1. 문제 은행 → Part 선택
2. 문제 추가 버튼 클릭
3. 파트별 필수 항목 입력
4. 시간 조정 (필요시)
5. 등록 클릭

**최소 필요 문제 수:**
- Part 1: 2개 이상
- Part 2: 2개 이상
- Part 3: 3개 이상
- Part 4: 3개 이상
- Part 5: 1개 이상

### 3단계: 테스트 진행
1. 사용자가 로그인 (이름 + 수험번호)
2. 마이크/네트워크 체크
3. 테스트 시작
4. 자동 진행 및 녹음

### 4단계: 채점
1. 채점 → 응시자 선택
2. 11개 녹음 파일 청취
3. 점수 엑셀 업로드

### 5단계: 성적표 발급
1. 성적표 관리
2. PDF 다운로드

## 🎨 새로운 기능 하이라이트

### ✨ Toast 알림 시스템
- 우측 상단에 실시간 알림 표시
- 성공(초록), 오류(빨강), 정보(파랑)
- 자동으로 사라짐

### 📊 진행 상황 실시간 표시
- "파일 읽는 중..."
- "10명의 사용자 데이터 전송 중..."
- "업로드 완료!"

### 🎯 파트별 맞춤 UI
- 각 파트 특성에 맞는 입력 폼
- 자동 시간 설정 (수정 가능)
- 파일 타입별 업로드 (음성/이미지)

### ⏱️ 유연한 시간 설정
- 준비 시간: 0~180초
- 말하기 시간: 15~120초
- 문제별 개별 설정 가능


## 🔍 Supabase에서 확인

1. Supabase 대시보드 접속
2. Table Editor 클릭
3. 테이블 확인:
   - `users` - 응시자 목록
   - `admins` - 관리자 목록
   - `questions` - 문제 은행
   - `test_attempts` - 응시 기록
   - `recordings` - 녹음 파일
   - `scores` - 점수 및 성적
